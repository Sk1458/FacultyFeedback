User Manual for NLP-driven Sentiment Analysis-based Faculty Insights System

Introduction

This system provides insights on faculty performance using sentiment analysis of feedback provided by students. It utilizes Natural Language Processing (NLP) to analyze textual data and deliver accurate insights to faculty members and administrators.

Prerequisites

  • Stable internet connection for accessing the system.
  • Login credentials (Username and Password) to access your account.
  • A browser that supports HTML5 and JavaScript (latest versions recommended).
  • Ensure that cookies are enabled in your browser for a smoother experience.

System Features

  • User-friendly interface for easy navigation.
  • Secure login and registration system.
  • Analysis of student feedback using sentiment analysis.
  • Clear and understandable visual insights for faculty members.

How to Use

Follow these steps to start using the system:

  • Go to the homepage and click on the Login button.
  • Enter your username and password.
  • If you're a new user, click on Register to create an account.

    Note: Registration is only available for students pursuing a BTech in CSE at Aditya College of Engineering and Technology.

  • After logging in, you will have access to comprehensive insights into faculty performance, as well as the opportunity to provide feedback on individual faculty members.
